도커는 사실 쓰는 법을 몰라 살짝 애먹었다..

Dreamhack문제에서 deploy, Dockerfile을 다운 받고
그 경로에서 docker build -t 내도커아이디(이름)/파일이름:태그 .

이런식으로 이지를 먼저 만들었다
.은 현재 경로를 뜻한다

그리고

docker run -it 이미지아이디 /bin/bash
를 입력하여 컨테이너를 킨다.

/bin/bash을 같이 입력해줘야 직접 터미널을 조작할 수 있다.

ls 명령어로 무슨 파일이 있는지 확인을 해보았더니 flag 파일을 발견하였고 이전과 동일하게 cat flag를 하여 flag를 획득하였다.
